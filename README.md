## Redactify

The source for Redactify at http://redactify.com

### Installation
* config/twitter_auth_example.yml to config/twitter_auth.yml 
* Obtain a API key from Twitter at http://twitter.com/apps
* Put the values in config/twitter_auth.yml
* rake db:migrate
